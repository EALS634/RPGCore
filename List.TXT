Minecraft - Pocket Edition Nukkit Plugin RPG Project

00 : RPG 기본 체계 구현
 - EXP : 레벨업에 필요한 시스템 [ 경험치량 식 : 레벨 X 100 ]
  - EXP : 몬스터나 유저를 죽일시 경험치 알갱이들이 일정 드롭 보상 [ 흡수 ] 
 - LV : 레벨에 따른 보상 및 활용도 증가 시스템
  - 레벨이 오르면 그에 따른 보상 [ 아이템, EXP, 버프 등 ]
 - 스탯 : 능력치를 조정 합니다.
  - 레벨업시 필요한 스탯을 일정 수를 주고 공격/방어/~ 등 여러 능력치를 향상시키는데 이용
 - 일정 레벨이 될시 ( 인첸트 장비 등 [ 금, 은, 동 상자 ] 아이템을 지급함 [ 세트팩 ]
 - 일정 도전 ( 목표 )치가 도달할시 칭호 혹은 여러보상 지급
 - 레벨이 오르거나 목표를 달성시 플레이어 주변에 폭죽 같은 효과 발동
 - 레벨업을 할시 일정 보상을 해줘야 합니다(추가:스킬 포인트, 골드보너스(퍼센테이지로보너스), 골드
 - 스테미나 ( SP ) 시스템 구현
  - 기술이나 달리기 등으로 기력을 써먹고 기력이 부족할시 충전 ( 충전은 자동 아니면 수동으로 충전 )
 - 무기강화 : 인첸트 같은 시스템입니다.
 - 자신이 킬/데스가 일정수치가 되면 게임머니/아이템/버프/EXP 등 보상을 받습니다.
 - 장비에 내구도가 달아 수리를 해야 이용이 가능하게 합니다.
 - 게임내에 경제 게임머니를 원할한 작동을 위해 세금을 적용합니다. [ 시간당 소지금의 1% 혹은 시간당 1만 달러의 세금 ], [ 접속하는 시간만 세금 작동 ]
 - 플레이 시간에 따른 보상
 - 콜로세움 같은 대규모 PVP장소에 유저들을 싸그리 몰아너어 최종의 1인 혹은 랭커에 오른 유저 보상
 - 경제 같은 인플리엔션 ( 경제 버그 등 ) 해결방안
  - 도박 ( 아이템, 게임머니, 수리, 등... )
 - 던전 시스템 ( 맵에 몬스터가 스폰되고 일정 몬스터를 깨면 보상을 주는 시스템 )
  - 일반, 보스, 레이드, PVP, 공성전, 스카이블럭, 헝거게임 ~....
 - RPGCore 에 들어가야하는 Function 적 내용
  - 팁체계에다 상속클래스로 표출력하는 기능
  - 눈싸움 등..
  - 특정 아이템에 효과를 추가하여 플레이어에게 데미지를 줌 ( ex. 번개가 떨어지고 밴당한다던지..등... )
  - 모루 코드 조금 응용해서 아이템 이름 바꿔넣는거 개발
  - 유저를 공격하는데 ~한 아이템을 들고 점프버튼을 누르면 3단 점프가 되서 공격할 수 있는 함수
  - ~한 상태(레벨이나 칭호획득)가 되었을때 유저 주위로 파티클로 오오라를 넣어줌
  - 내구도 조절 
  - 불화살, 얼음 화살, 독화살 등 포션효과가 붙은 투척용 무기
  -
